# Mathematical Modeling Coding

This repository contains the code, input data, intermediate tables, figures, and generated results for the B problem modeling workflow.

## Structure

- `src/`: core modeling, alignment, fusion, diagnostics, and task optimization modules.
- `scripts/`: reproducibility entry scripts.
- `outputs/`: generated trajectories, tables, figures, and decision logs.
- `附件*.xlsx`: original input data files.
- `result.xlsx` and `outputs/result_filled.xlsx`: result workbook files.

The paper/writing source is intentionally kept out of this repository.
